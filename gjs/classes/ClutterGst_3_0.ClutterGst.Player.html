set_playing

  • set_playing(playing: boolean): void
    • Defined in ClutterGst-3.0.d.ts:142

    Starts or stops playing of self.

    The implementation might be asynchronous, so the way to know whether the actual playing state of the self is to use the #GObject::notify signal on the #ClutterGstPlayer:playing property and then retrieve the current state with clutter_gst_player_get_playing(). ClutterGstVideoActor in clutter-gst is an example of such an asynchronous implementation.

    Parameters

    • playing: boolean

    Returns void
